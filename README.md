## Loops divided by three types
* ## Loop for
### for (let i=0 ; i<=10 ; i++)
### {
    console.log(i)
}

* ## Loop While
### let i = 0
### While (i <= 10)
### {
    i++
    console.log(i)
}

* ## Loop Do/While
### let i = 0
### DO
### {
    i++
    console.log(i)
} While(i <= 10)
